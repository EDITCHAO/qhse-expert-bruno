# ⚡ Next.js + Vercel - Démarrage Rapide (10 minutes)

## 🎯 Objectif
Déployer votre portfolio Next.js sur Vercel

---

## 📋 Checklist Rapide

### 1️⃣ Pousser sur GitHub (2 min)
```bash
git add frontend-nextjs/
git commit -m "Préparer Next.js pour Vercel"
git push origin main
```

### 2️⃣ Créer Projet Vercel (3 min)
```
1. Allez sur: https://vercel.com/new
2. Sélectionnez: EDITCHAO/qhse-expert-bruno
3. Framework: Next.js (auto-détecté)
4. Root Directory: frontend-nextjs
5. Environment Variables:
   - NEXT_PUBLIC_API_URL = http://localhost:8000
6. Cliquez: Deploy
```

### 3️⃣ Attendre Déploiement (3 min)
```
Vercel construit et déploie
Vous recevez une URL
```

### 4️⃣ Tester (2 min)
```
1. Allez sur: https://qhse-expert-bruno.vercel.app
2. Vérifiez que tout fonctionne
3. Testez sur mobile
```

---

## 🎉 Résultat

**Votre site est en ligne!**
