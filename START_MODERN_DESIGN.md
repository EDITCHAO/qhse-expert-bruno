### 1️⃣ Installation
```bash

npm install
```

### 2️⃣ Démarrage
```bash
npm start
```

### 3️⃣ Voir le site
Ouvrez `http://localhost:3000`

## 📸 Ajouter votre photo

Modifiez `src/components/About.js`:
```javascript
image: 'VOTRE_IMAGE_URL'
```

## 🔧 Personnaliser

Éditez `src/config/siteConfig.js`:
```javascript
profile: {
  name: 'VOTRE NOM',
  title: 'VOTRE TITRE',
  image: 'VOTRE_IMAGE_URL'
}
```
