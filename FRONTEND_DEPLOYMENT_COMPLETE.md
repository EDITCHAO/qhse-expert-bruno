## 🚀 Déploiement en 4 Étapes

### Étape 1: Créer Compte Vercel
```
Durée: 2 minutes
URL: https://vercel.com
Action: Sign Up → GitHub
```

### Étape 2: Importer Repository
```
Durée: 3 minutes
URL: https://vercel.com/new
Action: Import Git Repository
Repository: https://github.com/EDITCHAO/qhse-expert-bruno.git
```

### Étape 3: Configurer Projet
```
Durée: 3 minutes
Project Name: qhse-expert-bruno
Framework: Create React App
Root Directory: frontend
Environment Variables:
  - CI = false
  - REACT_APP_API_URL = http://localhost:8000
```

### Étape 4: Déployer
```
Durée: 2 minutes
Action: Cliquez "Deploy"
Résultat: https://qhse-expert-bruno.vercel.app
```
