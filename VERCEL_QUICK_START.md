# ⚡ Vercel - Démarrage Rapide (10 minutes)

## 🎯 Objectif
Mettre votre frontend en ligne sur Vercel

---

## 📋 Checklist Rapide

### 1️⃣ Créer Compte Vercel (2 min)
- [ ] Allez sur: https://vercel.com
- [ ] Cliquez: "Sign Up"
- [ ] Connectez-vous avec GitHub
- [ ] Confirmez votre email

### 2️⃣ Importer Repository (3 min)
- [ ] Allez sur: https://vercel.com/new
- [ ] Sélectionnez: "Import Git Repository"
- [ ] Entrez: `https://github.com/EDITCHAO/qhse-expert-bruno.git`
- [ ] Cliquez: "Continue"

### 3️⃣ Configurer Projet (3 min)
- [ ] Project Name: `qhse-expert-bruno`
- [ ] Framework: **Create React App**
- [ ] Root Directory: **frontend**
- [ ] Environment Variables:
  ```
  CI = false
  REACT_APP_API_URL = http://localhost:8000
  ```

### 4️⃣ Déployer (2 min)
- [ ] Cliquez: "Deploy"
- [ ] Attendez la fin du déploiement
- [ ] Vous recevrez une URL: `https://qhse-expert-bruno.vercel.app`

---

## 🎉 Résultat

✅ **Votre site est en ligne!**

**URL:** https://qhse-expert-bruno.vercel.app
