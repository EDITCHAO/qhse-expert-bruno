## 🚀 Déploiement en 4 Étapes (10 minutes)

### 1️⃣ Créer Compte Vercel (2 min)
```
https://vercel.com → Sign Up → GitHub
```

### 2️⃣ Importer Repository (3 min)
```
https://vercel.com/new → Import Git Repository
https://github.com/EDITCHAO/qhse-expert-bruno.git
```

### 3️⃣ Configurer Projet (3 min)
```
Project Name: qhse-expert-bruno
Framework: Create React App
Root Directory: frontend
Environment Variables:
  - CI = false
  - REACT_APP_API_URL = http://localhost:8000
```

### 4️⃣ Déployer (2 min)
```
Cliquez: Deploy
Attendez la fin
Votre URL: https://qhse-expert-bruno.vercel.app
```
