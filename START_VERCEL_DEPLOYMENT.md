## ⚡ Déploiement Rapide (10 minutes)

### Étape 1: Créer Compte Vercel (2 min)
```
1. Allez sur: https://vercel.com
2. Cliquez: "Sign Up"
3. Connectez-vous avec GitHub
4. Confirmez votre email
```

### Étape 2: Importer Repository (3 min)
```
1. Allez sur: https://vercel.com/new
2. Sélectionnez: "Import Git Repository"
3. Entrez: https://github.com/EDITCHAO/qhse-expert-bruno.git
4. Cliquez: "Continue"
```

### Étape 3: Configurer Projet (3 min)
```
1. Project Name: qhse-expert-bruno
2. Framework: Create React App
3. Root Directory: frontend
4. Environment Variables:
   - CI = false
   - REACT_APP_API_URL = http://localhost:8000
5. Cliquez: "Deploy"
```

### Étape 4: Attendre Déploiement (2 min)
```
1. Vercel construit votre site
2. Vercel déploie automatiquement
3. Vous recevez une URL: https://qhse-expert-bruno.vercel.app
```
