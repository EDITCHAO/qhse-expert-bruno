## 🚀 Déploiement en 3 Étapes

### Étape 1: Pousser sur GitHub
```bash
git add frontend-nextjs/
git commit -m "Préparer Next.js pour Vercel"
git push origin main
```

### Étape 2: Créer Projet Vercel
```
1. Allez sur: https://vercel.com/new
2. Sélectionnez: EDITCHAO/qhse-expert-bruno
3. Framework: Next.js (détecté automatiquement)
4. Root Directory: frontend-nextjs
5. Environment Variables:
   - NEXT_PUBLIC_API_URL = http://localhost:8000
6. Cliquez: Deploy
```

### Étape 3: Attendre Déploiement
```
Vercel construit et déploie automatiquement
Vous recevez une URL: https://qhse-expert-bruno.vercel.app
```
