                        TROUBLESHOOTING
================================================================================

Build Fails:
   • Check Vercel logs
   • Run "npm run build" locally
   • Fix errors and push to GitHub

Site Shows 404:
   • Verify Output Directory is "build"
   • Check build completed successfully
   • Clear browser cache

Variables Not Working:
   • Verify variables added in Vercel
   • Redeploy after adding variables
   • Check names start with "REACT_APP_"
