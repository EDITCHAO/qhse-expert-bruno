## 🚀 Démarrage

```bash
cd frontend
npm install
npm start
```

## 📸 Ajouter votre photo

Modifiez `src/components/About.js`:
```javascript
image: 'VOTRE_IMAGE_URL'
```

## 🔧 Personnaliser

Éditez `src/config/siteConfig.js`:
```javascript
profile: {
  name: 'VOTRE NOM',
  title: 'VOTRE TITRE',
  image: 'VOTRE_IMAGE_URL'
}
```
