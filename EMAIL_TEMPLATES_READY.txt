================================================================================
                        ÉTAPES DE CONFIGURATION
================================================================================

1. INSTALLER NODEMAILER
   npm install nodemailer

2. GÉNÉRER MOT DE PASSE GMAIL
   https://myaccount.google.com/apppasswords

3. CONFIGURER .env
   EMAIL_USER=editchaosam@gmail.com
   EMAIL_PASSWORD=xxxx xxxx xxxx xxxx

4. REDÉMARRER BACKEND
   npm run dev

5. TESTER
   - Remplissez le formulaire
   - Vérifiez les emails reçus

================================================================================
                        TESTER LES EMAILS
================================================================================

1. ALLER SUR LE SITE
   http://localhost:3000

2. REMPLIR LE FORMULAIRE
   - Nom: Votre nom
   - Email: Votre email
   - Téléphone: Votre numéro
   - Sujet: Un sujet
   - Message: Un message

3. ENVOYER
   Cliquez sur "Envoyer"

4. VÉRIFIER
   - Vérifiez que vous recevez l'email à editchaosam@gmail.com
   - Vérifiez que le client reçoit la confirmation
   - Vérifiez la date et l'heure
   - Vérifiez la structure et le formatage
