## 🔧 Personnaliser

### Modifier les projets
Éditez `src/components/Projects.js`:
```javascript
const projects = [
  {
    id: 1,
    title: 'Votre projet',
    category: 'Catégorie',
    impact: 'Impact',
    stats: { key1: 'value1', key2: 'value2', key3: 'value3' },
    icon: 'fa-icon',
    color: 'color'
  }
]
```

### Modifier les logos
Éditez `src/components/Logos.js`:
```javascript
const logos = [
  { id: 1, name: 'Nom', icon: 'fa-icon', color: 'color' }
]
```
