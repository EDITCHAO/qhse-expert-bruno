
### 3. Ajouter votre photo

## 📸 Photo de profil

### Où modifier
- `src/components/About.js` (ligne ~15)
- `src/components/Profiles.js` (ligne ~12)

### Options
1. **Fichier local**: `frontend/public/images/profile.jpg`
2. **URL externe**: `https://...`
3. **Import module**: `import profileImage from '...'`

## 🔧 Personnalisation

### Modifier les données
Éditez `src/config/siteConfig.js`:
```javascript
profile: {
  name: 'VOTRE NOM',
  title: 'VOTRE TITRE',
  description: 'VOTRE DESCRIPTION',
  image: 'VOTRE_IMAGE_URL'
}
```
