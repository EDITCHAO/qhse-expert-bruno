## 🚀 Démarrage rapide

### 1. Installation
```bash
cd frontend
npm install
```

### 2. Démarrage
```bash
npm start
```

### 3. Voir le site
Ouvrez `http://localhost:3000`

## 📸 Ajouter votre photo

Modifiez `src/components/About.js`:
```javascript
image: 'VOTRE_IMAGE_URL'
```

## 🔧 Personnalisation

### Modifier le profil
Éditez `src/config/siteConfig.js`:
```javascript
profile: {
  name: 'VOTRE NOM',
  title: 'VOTRE TITRE',
  image: 'VOTRE_IMAGE_URL'
}
```

### Modifier les services
```javascript
services: [
  {
    icon: '🔍',
    title: 'Service',
    description: 'Description',
    features: ['Feature 1', 'Feature 2']
  }
]
```
