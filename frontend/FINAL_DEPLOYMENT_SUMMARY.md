## 🆘 TROUBLESHOOTING

### Build Fails
```
1. Check Vercel logs
2. Run "npm run build" locally
3. Fix errors and push to GitHub
```

### Site Shows 404
```
1. Verify Output Directory is "build"
2. Check build completed successfully
3. Clear browser cache
```

### Variables Not Working
```
1. Verify variables added in Vercel
2. Redeploy after adding variables
3. Check names start with "REACT_APP_"
```
