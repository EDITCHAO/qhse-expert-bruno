================================================================================
                    WHAT HAPPENS DURING DEPLOYMENT
================================================================================

Step 1: Vercel Receives Your Project
        ↓
        • Clones from GitHub
        • Reads vercel.json
        • Reads package.json

Step 2: Install Dependencies
        ↓
        • Runs: npm install
        • Downloads all packages
        • Creates node_modules

Step 3: Build Application
        ↓
        • Runs: npm run build
        • Compiles React code
        • Optimizes assets
        • Creates build/ folder

Step 4: Optimize & Deploy
        ↓
        • Minifies CSS & JavaScript
        • Compresses images
        • Uploads to CDN
        • Configures routing

Step 5: Go Live
        ↓
        • Assigns URL
        • Enables HTTPS
        • Activates site
        • Ready for visitors!
